# 🎯 Decision Spinner

A futuristic neon-themed decision spinner built using HTML, CSS, and JavaScript.  
Spin the wheel and let the cosmos decide your fate.

---

## ✨ Features

- 🎡 Smooth animated spinning wheel
- ⚡ Neon cyberpunk UI
- 🔊 Built-in sound effects using Web Audio API
- 🎉 Confetti celebration on winner
- 💾 LocalStorage support
- 📜 Spin history tracking
- 📱 Fully responsive design
- 🌈 Custom option colors
- 🪄 Glassmorphism popup effects

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Canvas API
- Web Audio API
